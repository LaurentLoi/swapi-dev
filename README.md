# Another Swapi-Dev

Swapi (The Star Wars API) TypeScript/Node.js exercise.

Forked from https://gitlab.qualif.io/interview/swapi-dev.

Check [Swapi-dev README](README.legacy.md) for exercise instructions.

## How to run:

Clone the repo & run a cmd line.

- > $ npm i

---

#### CLI:

- > $ npm start

Or:

- > $ npm run compile

- > $ node dist/index.js
